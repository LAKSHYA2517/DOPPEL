# Twin Agent Frontend 🖥️

The UI for the Twin Agent. Designed to be performant, responsive, and aesthetically engaging.

## 🚀 Setup Instructions

1. Navigate to this directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```
   The site will be available at `http://localhost:5173`. Make sure the python backend is running simultaneously on port `8000`.
